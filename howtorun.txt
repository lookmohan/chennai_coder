# Chennai Coder - Website Run Guide

This project is a premium, high-performance portfolio and business website designed for Chennai Coder (Mohanraj).

---

## 🛠️ Technologies Used

1. **Frontend Core**: React 18 & TypeScript
2. **Scaffolding/Bundler**: Vite (super-fast hot module replacement & production builds)
3. **Styling**: Modern CSS modules with custom design tokens (light/dark variables, glassmorphism, responsive utilities)
4. **Icons**: Lucide React (vector-based clean design icons)
5. **Assets**: 
   - Authentic professional portrait: `public/assets/mohanraj-profile.jpg`
   - Official company logo: `public/assets/chennai-coder-logo.png`
6. **SEO & Metadata**: 
   - OpenGraph and Twitter card configuration for rich link preview shares.
   - Structured JSON-LD metadata for Person, Organization, and ProfessionalService schemas.
   - `robots.txt` and sitemap parameters.

---

## 🚀 How to Run the Project Locally

### Prerequisites
Make sure you have **Node.js** (v18.x or higher) installed on your system.

### 1. Install Dependencies
Open your terminal at the root of the `chennai-coder` project directory and run:
```bash
npm install
```

### 2. Run the Development Server
Launch the live local development server with:
```bash
npm run dev
```
Once started, open your browser and navigate to the local address displayed:
👉 **http://localhost:5173/**

### 3. Build for Production
To compile and optimize the website for production deployment (creating a `dist/` folder):
```bash
npm run build
```

### 4. Preview the Production Build
To run a local server previewing the compiled production bundle:
```bash
npm run preview
```
